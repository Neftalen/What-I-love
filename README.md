# 🌍 My 3D Printing World

A simple multi-page website about my interest in 3D printing and creativity.

## 📌 About The Project

This website is a personal project where I share:

* What 3D printing is
* How it workst
* My favorite 3D prints
* Ideas I want to create in the future

Even though I don’t own a 3D printer yet, I enjoy learning and watching the process of turning digital designs into real objects.
